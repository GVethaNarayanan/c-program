# 💻 C Programming Practice Repository

## 🚀 Overview

This repository contains a collection of C programming programs ranging from basic to intermediate level. It is designed to strengthen problem-solving skills and improve understanding of core programming concepts.

---

## 🎯 Objectives

* Learn and practice C programming fundamentals
* Improve logic-building and problem-solving skills
* Prepare for coding interviews and exams
* Build a strong foundation in programming

---

## 📂 Topics Covered

* 🔢 Basic Programs (Hello World, Patterns, etc.)
* 🔁 Loops (for, while, do-while)
* 🔀 Conditional Statements (if-else, switch)
* 📊 Arrays & Strings
* 🔢 Functions
* 📦 Pointers
* 📁 File Handling
* 🧮 Mathematical Programs

---

## 📁 Project Structure

```
c-program/
│── basics/            # Basic C programs

│── arrays/            # Array programs
│── strings/           # String manipulation
│── functions/         # Function-based programs
│── pointers/          # Pointer concepts
│── files/             # File handling programs
```

---

## ▶️ How to Run

1. Install a C compiler (like GCC)
2. Compile the program:

```
gcc program.c -o output
```

3. Run the executable:

```
./output
```
